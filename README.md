# AbsImm-Xmpp

A XMPP client for AbsImm

## Requirements

Build:
* Java JDK >= 1.7
* Gradle >= 2.2

Run:
* Java JRE >= 1.7

## Build

Use Gradle to build the project:

        gradle distZip

## Run

Extract the zip file and run:

        java -jar bin/absimm-xmpp <storyfile> <user> <password> <host> <port>
